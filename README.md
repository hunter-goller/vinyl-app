# Vinyl App 🎵

A personal vinyl collection + listening tracker built with React Native.

## Goals
- Track vinyl collection
- Log plays (manually + NFC tap)
- View listening stats
- Get recommendations based on listening history

## Tech Stack
- React Native (Expo)
- TypeScript
- SQLite (local database)

## MVP Features
- Add / edit / delete albums
- Track play history
- NFC tap-to-log plays
- Basic statistics
